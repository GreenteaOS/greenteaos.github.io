About Greentea OS

Warm and cozy, open-source operating system compatible with Windows apps, games and drivers

The main idea of the project is to quickly and efficiently create a compatible environment for the existing software and provide the best user experience of a personal computer.

There is no compatible Windows alternatives. Someone should be first. We are the first. Our project — Greentea OS — aims at making familiar user interface, running existing apps and make you a happy computer user. We are not here yet, but we are not going to stop.
